# ip4player
## A simple library and Placeholder api.

---

**usage**:

```
%ip4player_ip% : get player's ip.
%ip4player_country%: get player's country.
%ip4player_province%: get player's province(if available).
%ip4player_city%: get player's city(if available).
%ip4player_isp%: get player's ISP.
%ip4player_region%: formatted full string of region information.
```
